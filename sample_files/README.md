# Sample Files

Place media files here for testing.

## Supported Formats

### Audio
- `sample_audio.mp3` - MP3 file
- `sample_audio.wav` - WAV file

### Video
- `sample_video.mp4` - MP4 file

### Images
- `sample_image.jpg` - JPG image with text
- `sample_image.png` - PNG image with text

## Generate Sample Audio

```bash
python sample_files/create_samples.py
